# Déjà Vue — Memory Match

A card-matching memory game built with React (SEG3125 Assignment 3).

## Run it locally

You need Node.js installed (https://nodejs.org). Then, in this folder:

    npm install
    npm run dev

Open the URL it prints (usually http://localhost:5173).

## Build for deployment

    npm run build

This creates a `dist/` folder. To deploy on Vercel, push this folder to
GitHub and import the repo. Vercel auto-detects Vite:
build command `npm run build`, output directory `dist`.
